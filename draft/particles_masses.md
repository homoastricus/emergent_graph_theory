## Эмерджентные массы элементарных частиц в модели графа малого мира

### Структурные функции
- f₁ = U/π
- f₂ = ln K
- f₃ = √(K p)
- f₄ = 1/p
- f₅ = K/ln K
- f₆ = (K + pK)/K

где U = ln N / |ln(K p)|


**Базовые константы:**
- `K` - степень вершин графа
- `p` - параметр вероятности связи
- `N` - число вершин графа

## Примечания

1. Все массы рассчитываются относительно базовой массы электрона: `m_e = 1` (в соответствующих единицах)
2. Структурные функции вычисляются на основе параметров графа `K`, `p`, `N`
3. Ошибка показывает процентное расхождение между экспериментальными и теоретическими значениями
4. Формулы показывают комбинации структурных функций, необходимые для расчета массы каждой частицы
